Referring now to FIG. 1, illustrative cloud computing environment 102 is depicted. As shown, cloud computing environment 102 includes one or more cloud computing nodes 104 with which local computing devices used by cloud consumers, such as, for example, personal digital assistant (PDA) or cellular telephone 106, desktop computer 108, laptop computer 110, and/or automobile computer system 112 may communicate. computing nodes 104 may communicate with one another. They may be grouped (not shown) physically or virtually, in one or more networks, such as Private, Community, Public, or Hybrid clouds as described hereinabove, or a combination thereof. This allows cloud computing environment 102 to offer infrastructure, platforms and/or software as services for which a cloud consumer does not need to maintain resources on a local computing device. It is understood that the types of computing devices  shown in FIG. 1 are intended to be illustrative only and that computing nodes 104 and cloud computing environment 102 can communicate with any type of computerized device over any type of network and/or network addressable connection (e.g., using a web browser).
